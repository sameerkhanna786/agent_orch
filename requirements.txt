# --- pydantic ABI pin (F1) ---
# pydantic ships a Rust extension (pydantic-core) whose ABI is pinned per pydantic
# minor. A stale pydantic-core (e.g. 2.20.1 left from a partial venv rebuild) against
# a newer pydantic crashes `import datasets` in repo prep, taking down EVERY eval cell.
# Keep this in lockstep with PYDANTIC_CORE_PIN in apex_omega/eval/commit0_autogen.py
# (which also runs a loud preflight check). pydantic 2.11.x is built against
# pydantic-core 2.46.1; if you bump pydantic, bump BOTH pins together.
